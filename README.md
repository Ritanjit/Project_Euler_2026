# Project Euler — C++ Solutions

My solutions to [Project Euler](https://projecteuler.net/) problems, written in C++.

## Problems Solved

| # | Problem | Concepts |
|---|---------|----------|
| 1 | [Multiples of 3 or 5](https://projecteuler.net/problem=1) | Iteration, modular arithmetic |
| 2 | [Even Fibonacci Numbers](https://projecteuler.net/problem=2) | Fibonacci sequence, parity filtering |
| 3 | [Largest Prime Factor](https://projecteuler.net/problem=3) | Prime factorization, trial division |
| 4 | [Largest Palindrome Product](https://projecteuler.net/problem=4) | Palindrome check, brute-force optimization |
| 5 | [Smallest Multiple](https://projecteuler.net/problem=5) | Sieve of Eratosthenes, LCM via prime powers |
| 6 | [Sum Square Difference](https://projecteuler.net/problem=6) | *(in progress)* |

## Build & Run

Requires a C++ compiler (g++, MSVC, clang, etc.).

```bash
g++ P1.cpp -o P1 && ./P1
```
